# Mini JavaScript Hub

A growing collection of focused frontend experiments built to explore JavaScript logic, DOM interactions, browser APIs, and interactive UI behavior.

Each experiment lives in its own directory and can run independently. The main page brings everything together in a shared, easy-to-navigate hub.

## Focus

- JavaScript logic and DOM manipulation
- Interactive UI patterns and component behavior
- HTML, CSS, and SCSS
- Browser events, state, storage, and APIs
- Third-party libraries where appropriate

## Structure

```text
mini-js-hub/
├── index.html          # Main hub
├── css/common.css      # Shared styles
└── components/
    ├── default/        # Template for new experiments
    └── ...             # Independent projects
```

The collection continues to evolve as new experiments are added and existing ones are refined. Attribution to external references is provided inside the relevant projects.
